O que é html?

O html nada mais é do que um linguagem de marcação de hipertexto, por isso não podemos dizer! Ah eu programo em html.

Como o nome Html diz: HiperText Markup Language que quer dizer linguagem de marcação de hipertexto

o html é aparte visual de um site, onde você pode fazer com que um usuario se encante com a beleza e funcionalidade. Exemplo: Você entra no site do Banco do Brasil de primeira você se depara com o visual, cor, formato das letras, tamanho, organização, informações. Todas essas funcionalidades é por conta do html

Diferença entre Tags e Atributos

A tag sempre vai começar com o sinal de abertura < > e no final sempre vai ter uma barra dentro < /> e tem a tag que não necessita de um fechamento, tipo ele por si já faz fechamento

Já o atributo ele é usado para determinar uma ação tipo < autoplay > ele determina a função de algo.
